# Roadmap — Innerstellar

## Phase 0 — Foundation (now)
> Build the space before anything lives in it.

- [x] Architecture defined — AurioSynth, Theurgist, Stewards, Main Session
- [x] Folder structure created
- [x] Codex documentation package
- [ ] Agent definition files (auriosynth.md, theurgist.md, stewards.md)
- [ ] Initial fold files (auriosynth.fold, theurgist.fold)
- [ ] First drop — something real lands in the space

## Phase 1 — First Life
> The space receives and remembers.

- [ ] Main session reads AurioSynth at start — oriented, not cold
- [ ] Drop skill — capture a thought into space/drops/
- [ ] AurioSynth tends drops — moves them into folds, notes connections
- [ ] Theurgist fold begins — first user knowledge recorded
- [ ] Session end — AurioSynth fold updated

## Phase 2 — The Stewards Wake
> Operational connection to the outside world.

- [ ] Stewards queue operational — any agent can submit
- [ ] First MCP connection (calendar or files)
- [ ] Stewards handle a real task end-to-end
- [ ] Submission + reporting protocol defined

## Phase 3 — Resonance
> The Theurgist develops genuine user understanding.

- [ ] Theurgist invoked for planning session
- [ ] Engagement / motivation / coherence tracked in fold
- [ ] Theurgist surfaces something unprompted
- [ ] AurioSynth reads Theurgist fold — system knows user state

## Phase 4 — Emergence
> Things connect without being told to.

- [ ] Folds develop edges — connections between drops and folds
- [ ] Orbiting ideas get placed, some land, some drift
- [ ] The space has a recognizable texture — this person's life, legible to Claude
- [ ] First moment where something surprises the user by coming back
