# Stewards Queue

System-level steward tasks. Cross-project, framework concerns, or tasks
not tied to a specific project.

## How It Works

Drop a task file here when a steward needs to act on something that belongs
to the framework rather than a specific project. Name it clearly:

```
YYYY-MM-DD_brief-description.md
```

Tasks tied to a specific project live in that project's own queue:
```
space/projects/{project-name}/queue/
```

## Task Format

```
---
submitted_by: [user | main_session | entity_name]
submitted: YYYY-MM-DD
steward: [social-media | email | calendar | files | media | archives]
priority: [now | soon | when_convenient]
status: [open | in_progress | done]
---

## Task

[what needs to happen]

## Output

[what done looks like]

## Notes

[anything the steward should know]
```
